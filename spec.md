# 数独生成器 Spec

## 功能需求
1. 生成完整终盘（随机但符合数独规则）。
2. 根据难度挖空（简单: 40空, 中等: 50空, 困难: 60空），保证有唯一解。
3. 提供 Web 界面：9x9 网格，用户点击格子输入数字 1-9，Delete 清除。
4. 点按棋盘上数字时，高亮同数字
5. 下方添加1～9数字条，灰掉已经填满的数字
6. 实时检查冲突（同行/列/宫重复高亮红色）。
7. 按钮：“新游戏”（选择难度）、“检查谜题”、“求解”（AI 自动填入剩余数字）。
8. 界面美观（CSS Grid 或 Flex，圆角，浅色/深色主题可选）。

## 技术栈
- 纯 javascript，无需后端。
- 算法：回溯法生成终盘 + 唯一解验证（可用 DLX 或简单回溯）。

## 验收标准
- 每次新游戏生成的谜题有唯一解。
- 用户输入非法字符（非 1-9）被忽略。
- 求解按钮能在 <1 秒内完成。

## 额外要求
- 代码结构清晰，分为 `generator.js`、`solver.js`、`ui.js`。
- 添加必要的注释。
